Nível 2 · A peça que mata a alucinação de formato

Template + JSON Schema = decodificação restrita.

Slide 06 · Nível 2
Schema strict (extrato real do CoRRaDI)
{
  "impression": {
    "type": "object",
    "required": ["summary", "category"],
    "properties": {
      "summary":     { "type": "string", "maxLength": 600 },
      "lung_rads":   { "enum": ["1","2","3","4A","4B","4X"] },
      "bi_rads":     { "enum": ["0","1","2","3","4A","4B","4C","5","6"] },
      "ti_rads":     { "enum": ["TR1","TR2","TR3","TR4","TR5"] },
      "laterality":  { "enum": ["direita","esquerda","bilateral"] },
      "recommendation": {
        "enum": ["none","follow_up_3m","follow_up_6m","biopsy","mdt"]
      }
    },
    "additionalProperties": false
  }
}
O modelo não pode emitir “Lung-RADS 4D”.
A decodificação recusa o token inválido. Erro de formato vira impossibilidade matemática.

Nível 3 · Onde a discussão fica adulta

RAG estratificado · estilo · fato · regra.

Slide 07 · Nível 3
1
Retriever de estilo

Laudos similares

Servem como exemplo de estilo: terminologia, ordem das seções, tom.
Não são fonte de fato sobre o paciente atual.
Index: embeddings de impressões anonimizadas do próprio serviço.
2
Retriever de fato

Laudos prévios do mesmo paciente

Comparativo factual: “nódulo estável em relação a 2024”.
Identificado por MRN, não por similaridade.
Index: filtro determinístico por paciente + recência.
3
Retriever de regra

Guidelines

Fleischner · BI-RADS · LI-RADS · PI-RADS · TI-RADS · ACR Appropriateness.
Trecho normativo, não opinião — vai pra system prompt.
Index: chunks autorais com versão e DOI.
Anti-pattern frequente
Misturar os três sob a mesma chave few_shots → alucinação cross-paciente: o LLM importa achado de exemplo similar e cola no caso atual.
Evidência
Tozuka 2024 — RAG ↓ alucinação em estadiamento TNM.
DIR 2025 — RAG ↑ acurácia em respostas BI-RADS.
Grounded MM-RAG — Recall@5 > 0.95 em retrieval radiológico.
Lewis et al., NeurIPS 2020 · Tozuka et al., Jpn J Radiol 2024 · Sun et al., DIR 2025 · Wang et al., “Grounded Multimodal RAG,” arXiv:2410.xxxxx.

Slide 08 · Nível 4

CGAFT · Adversarial Fine-Tuning
Radiologista atua como adversário — gera casos onde o modelo erra. O modelo é re-treinado nesses casos. Ciclo iterativo até hard-cases falharem < X%.

Grafo codifica regras radiológicas — “pneumonia ≠ osso”, “consolidação → ↑ densidade”. Tripleta inválida é bloqueada antes do texto sair.

Alucinação não desapareceu. Drift existe. Edição zero, não.

TarefaTaxaFonte

Tipos de alucinação descritos
Lesões fantasmas · anatomia errada · achados fabricados · números inventados · referência a estudo prévio inexistente.
Drift silencioso
GPT-3.5 · 38% → 26% (2024–25) · sem aviso. Bhayana, Radiology 2024.
Daí o model_full_id em runtime.
Vieses do LLM-juiz
→ self-preference — premia o próprio output.
→ verbosity bias — premia o mais longo.
→ position bias — premia o primeiro candidato.
Armadilha de design
Comando do usuário não pode ser prioridade absoluta. “Inclua ausência de trombose mesmo sem ter falado” = jailbreak. O sistema recusa ou marca para revisão.
VLM < texto · ainda em 2025
Input visual ainda alucina mais que textual. O degrau 5 é o mais alto, não o mais confiável.
A linha de fundo
Edição mínima — sim.
Edição zero — não.
O médico continua responsável.

Slide 18 · Regulação
NormaStatusO que exige · o que multa

ANPD Res. 19/2024 desde ago/2025 Cláusulas-padrão obrigatórias para transferir dados a APIs estrangeiras (OpenAI · Anthropic · Google · Microsoft). Sem isso → infração.
CFM 2.454/2026 vigência ago/2026 IA = apoio · médico decide · registro no prontuário · governança institucional formal.
PL 2.338 (Marco Legal IA) em tramitação Aprovado por unanimidade no Senado · dez/2024. Em tramitação na Câmara. Quando virar lei → hierarquia superior à resolução do CFM.
PL 2.688/2025 paralelo Proposta paralela em discussão.
ISO/IEC 42001 voluntária Norma internacional de governança de IA. Excelente evidência objetiva de boas práticas em auditoria.
Vedado

Permitido
API com BAA (Azure · Bedrock · Vertex) ou open-source on-premise.
